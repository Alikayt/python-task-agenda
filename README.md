# Python Task Agenda

This project is a simple task agenda built with Python.

It was created to organize daily tasks and keep track of activities in a clear and structured way, using a console-based interface.

## What this project does
- Allows users to add and view tasks
- Helps keep basic task information organized
- Focuses on clarity and ease of use

## Why this project
This agenda was developed as a practical exercise to apply Python fundamentals and basic logic, with a focus on real-world usability rather than complex features.

## Tools used
- Python

## Notes
This project runs in the console and does not include a graphical interface.
